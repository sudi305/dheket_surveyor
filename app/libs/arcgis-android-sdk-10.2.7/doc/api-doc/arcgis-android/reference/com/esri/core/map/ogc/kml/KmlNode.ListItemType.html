Class Overview

ListItemType specifies how a Feature is displayed in the list view or Table of content. Applicable only to Folder and Document KML nodes.

Enum Values

public static final KmlNode.ListItemType CHECK

This is default for all Nodes. Nodes visibility is associated with its item's checkbox.

public static final KmlNode.ListItemType CHECK_HIDE_CHILDREN

When specified use a normal checkbox for visibility but do not display the parent(container or Network Link) node's children in the list view or Table of content

public static final KmlNode.ListItemType CHECK_OFF_ONLY

When specified for a container node or a Network Link, should prevent all child nodes from being made visible at once. The user can turn all the child nodes off by selecting the check box for the container node or the Network Link, but cannot turn back all the child nodes on by selecting the parent checkbox.

public static final KmlNode.ListItemType RADIO_FOLDER

When specified for a container node, only one of the child nodes is visible at a time.
