Class Overview

The ArcGISLocalTiledLayer class is a type of tiled layer where the data is stored locally on the device, therefore this layer can function even when the device does not have any network connectivity. The data for this layer can be in the form of an ArcGIS Compact Cache or a tile package (.tpk file). The typical compact cache structure is as follows:
<CacheName>
     Layers
          _allLayers, conf.cdi, conf.xml
The path used in the constructor of the ArcGISLocalTiledLayer, when using an ArcGIS Compact Cache must point to the Layers folder e.g.
ArcGISLocalTiledLayer local = new ArcGISLocalTiledLayer("/sdcard/<CacheName>/Layers");
If using a tile package user needs to provide the full path, including the '.tpk' extension e.g.
ArcGISLocalTiledLayer local = new ArcGISLocalTiledLayer("/sdcard/<CacheName>.tpk");
