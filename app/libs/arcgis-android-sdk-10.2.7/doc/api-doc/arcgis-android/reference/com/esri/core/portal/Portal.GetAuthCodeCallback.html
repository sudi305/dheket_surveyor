public static interface

Portal.GetAuthCodeCallback

com.esri.core.portal.Portal.GetAuthCodeCallback

Class Overview

This is a callback interface used by the Portal.doOAuthUserAuthenticate() methods to get the calling app to display a web page that asks the user to login.

Summary

Public Methods
abstract String getAuthcode(String authorizationUrl)
The implementation of this method needs to display a web page that asks the user to login.

Public Methods

public abstract String getAuthcode (String authorizationUrl)

The implementation of this method needs to display a web page that asks the user to login. The authorization code resulting from a successful login is returned.

Parameters
authorizationUrl URL of login page
Returns
  • authorization code. Return null or empty String if login failed.
