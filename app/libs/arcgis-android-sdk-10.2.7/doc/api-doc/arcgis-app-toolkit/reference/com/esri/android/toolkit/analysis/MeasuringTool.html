Class Overview

The Measuring Tool follows Android's Contextual Action Bars design pattern to provide basic measuring and sketching capabilities in a self-contained tool. The tool can be readily plugged into your existing app provided the following two requirements are met:

  • An active MapView in your app and,
  • an Action Bar.
To wire the measure tool to your Action Bar, first create the MeasuringTool. Next, you need to create a new action button, which can usually be achieved by adding a MenuItem to the menu xml that populates the Action Bar. 

     <item
       android:id="@+id/action_measure"
       android:icon="@android:drawable/ic_menu_edit"
       android:orderInCategory="100"
       android:showAsAction="ifRoom"
       android:title="@string/measure"/>
Then add the following code snippet to your Activity or Fragment that contains the MapView to start the Measuring tool when the Action Button is clicked. 

 public boolean onOptionsItemSelected(MenuItem item) {
  switch (item.getItemId()) {
    case R.id.action_measure:
      startActionMode(new MeasuringTool(mapView));
      break;
    default:
      break;
  }
  return super.onOptionsItemSelected(item);
 }
The symbols used to draw lines and polygons can be customized by calling:
  • setMarkerSymbol(MarkerSymbol)
  • setLineSymbol(LineSymbol)
  • setFillSymbol(FillSymbol)
The linear and area units in the drop down list can also be customized by calling:
  • setLinearUnits(Unit[])
  • setAreaUnits(Unit[])
